### 1. 로컬 브랜치 이름 변경

```bash
git branch -m <old_branch_name> <new_branch_name>
```

### 2. 새 브랜치 원격 저장소에 push

```bash
git push origin <new_branch_name>
```

### 2. 원격 저장소에서 기존 브랜치 삭제

```bash
git push origin --delete <old_branch_name>
```
